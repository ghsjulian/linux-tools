# 🐧 Linux Tools

A powerful collection of useful Linux tools and scripts designed to simplify system management, automation, and terminal productivity.  
This repository provides lightweight and practical tools for developers, system administrators, and Linux enthusiasts.

## 📌 Overview

**linux-tools** is a curated toolkit that helps users perform common Linux operations quickly and efficiently through simple commands and scripts.  
The goal of this project is to provide easy-to-use, fast, and reliable utilities for everyday Linux workflows.

## 🛠 Installation

Clone the repository:

```bash
git clone https://github.com/ghsjulian/linux-tools.git
```

Go to project directory:

```bash
cd linux-tools
```

Give permission:

```bash
chmod +x *.sh
```

Run tools:

```bash
./tool-name.sh
```
